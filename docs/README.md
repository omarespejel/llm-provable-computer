# Documentation Map

This repository splits documentation into two surfaces:

- `docs/paper/`: publication-facing paper materials, appendices, evidence snapshots, and frozen cited artifact bundles
- `docs/engineering/`: implementation design notes, hardening policy, and broader reproducibility guidance for ongoing engineering work

For paper readers, start with:

- `docs/paper/README.md`

For trusted-core engineering work, start with:

- `docs/engineering/README.md`
