# Documentation Map

This repository splits documentation into three surfaces:

- [`docs/paper/`](paper/): publication-facing paper materials, appendices, evidence snapshots, and frozen cited artifact bundles
- [`docs/engineering/`](engineering/): implementation design notes, hardening policy, and broader reproducibility guidance for ongoing engineering work
- [`docs/security/`](security/): bounded threat-model and red-team matrix material for artifact binding, provenance, and backend-confusion risks

For paper readers, start with:

- [`docs/paper/README.md`](paper/README.md)

For trusted-core engineering work, start with:

- [`docs/engineering/README.md`](engineering/README.md)
- [`docs/engineering/ai-review-security-setup.md`](engineering/ai-review-security-setup.md)

For security and red-team review, start with:

- [`docs/security/threat-model.md`](security/threat-model.md)
- [`docs/security/red-team-matrix.md`](security/red-team-matrix.md)
